```markdown
# User Search Application - Back-end (Server)

This is the back-end part of the User Search Application. It serves as the server for handling user search requests.

## Prerequisites

Before running the back-end application, ensure you have the following dependency installed on your system:

- **Node.js:** [Download and Install Node.js](https://nodejs.org/)

## Setup

To run the back-end part of the application, follow these steps:

## 1 Install Dependencies:

     npm install

## 2 Start the Back-end Server:

     npm start

This command will start the back-end server, which will be accessible at http://localhost:3002.

Important: The back-end server must be running before starting the front-end. Please ensure that the front-end is also running as specified in its README. The front-end and back-end parts of the application work together.
```
